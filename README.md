
## ✨ Features

- **Responsive Design**: Fully responsive across all device sizes
- **Modern UI**: Neon-themed interface with smooth animations and intuitive navigation
- **Project Showcase**: Detailed project cards with filterable categories
- **Interactive Elements**: Dynamic navigation and hover effects
- **Performance Optimized**: Lazy loading images and minimal dependencies
- **Centralized Styling**: Global CSS variables for consistent theming
- **Modular JavaScript**: Separated functionality for better maintainability

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Custom CSS with CSS variables for theming
- **Architecture**: Mobile-first responsive design
- **Deployment**: Netlify
- **Version Control**: Git & GitHub

## 📋 Pages

- **Home**: Introduction with social media links (GitHub, LinkedIn, X, Netlify)
- **About**: Personal bio, background and skills
- **Projects**: Categorized and filterable project showcase
- **Resume**: Education, experience, certifications and technical skills
- **Contact**: Form to get in touch

## 📊 Skills Showcase

## 📱 Responsive Design

The portfolio is fully responsive with breakpoints at:
- Mobile: 375px and up
- Tablet: 768px and up
- Desktop: 992px and up
- Large Desktop: 1200px and up

## 🔧 Local Development

```powershell
# Clone the repository
git clone https://github.com/Nisanth-2025/portfolio.git

# Navigate to project directory
cd portfolio

# Open in browser
start index.html
```
